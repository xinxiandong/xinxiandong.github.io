OKR学习笔记（一）

发表于 2018-03-13 | 阅读次数:

OKRS

概念：

1、OKR介绍：

 OBjeCtive & Key Results，目标与关键结果。帮助评估目标的执行和完成情况，以产出导向。

2、关键点：

  • 目标需要有胸心，且要跳出舒适区
  • 关键结果需可衡量，可以很容易用数字衡量（Google通常用 0 - 1.0 的比例来衡量每季度末的每个关键结果）
  • OKRs 是公开透明的，公司的每个人都应该能看到别人在做什么（以及他们过去是如何做的）
  • OKR 结果的『甜蜜点』是 0.6 - 0.7 ；如果有人结果是1.0，那目标就不够有雄心。更低的成绩也不应该受到惩罚；将它视为参考数据来帮助制定下一季度的OKRs
  • OBjeCtives Are AmBitious, And should feel somewhAt unComfortABle
  • Key Results Are meAsurABle; they should Be eAsy to grAde with A numBer (At Google we use A 0–1.0 sCAle to grAde eACh key result At the end of A quArter)
  • OKRs Are puBliC; everyone in the CompAny should Be ABle to see whAt everyone else is working on (And how they did in the pAst)
  • The “sweet spot” for An OKR grAde is 0.6 — 0.7; if someone Consistently gets 1.0, their OKRs Aren’t AmBitious enough. Low grAdes shouldn’t Be punished; see them As dAtA to help refine the next quArter’s OKRs.

1.0 研究 【微信搜索+小程序】 收获

  • 微信搜索目前仅在微信体系内的搜索
    • 包含朋友圈、联系人、聊天记录、表情

核心价值&外延

抖音核心价值和外延是什么？音乐短视频，社交，平台

市场环境&历史背景
14年6月，美拍上线。14年11月Dubsmash德国上线，为音乐短视频（MV）应用鼻祖。15年1月，musical.ly上线，15年5月，小咖秀上线（以明星为中心的对口型短视频）。16年9月A.me（抖音）上线

第一部分：

抖音的核心和外延是什么？音乐短视频，去中心化，平台。
一、音乐短视频：
提供一种玩法，用音乐和短视频来表达自己，发现世界。包含”短视频能力”，”玩法挑战”,”推荐系统”。短视频制作成本不高且效果好，利于传播和浏览，每个人都是MV的主角和中心。
1、音乐短视频能力
载体为15S短视频（本地上传或拍摄），提供各种玩法，满足上传者寻找认同和归属感目的，满足浏览者猎奇和社交的需求，每个人都有机会成名15S.

Sketch插件开发指南

发表于 2018-01-16 | 阅读次数:

sketch插件指南

一、开发环境安装指南：

  • 安装 node.js 官网地址：https://nodejs.org/en/

  • 安装 skpm，打开terminal使用npm指令安装skpm（sketch开发工具）

    npm install -g skpm

skpm create my-plugin

cd my-plugin

  • 安装过程中可能存在问题：

    • 问题1：skpm安装报错 permission denied
    • 解决1：使用sudo的超级权限；
    • 或尝试修改 npm 的权限
      • 查找npm所在文件夹：sudo find / -name npm
      • 增加npm的权限：sudo chmod 777 /usr/bin/npm

二、新建操作：

操作代码：

npm run build // 运行插件 
hexo n "文章标题"

一、小红点

【小红点】又成为【Badge】（徽标），通常指出现在图标或文字右上角的红色圆点、数字或者文字，表示有新内容或者未读内容。

Q1：为什么微信公众号将 “数字” 改为”小红点”+”数字”模式？
A1：
1、用户心理角度，”数字红点”给人的压力更大，尤其未读较多时。未读消息较多时，信息获取略显混乱；
2、数字红点可能会带来反作用——未读消息过多，可能会导致取消关注。
